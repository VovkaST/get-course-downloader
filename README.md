# Плагин для скачивания видео с GetCourse

**Всегда помните, что любое распространение скачанных видео может являться нарушением авторских прав. Данное расширение создано исключительно для личного пользования.**

## Инструкция по установке и распространению

### 📦 Как упаковать расширение

Так как расширение не опубликовано в Chrome Web Store, его нужно передавать в виде папки или ZIP-архива.

1. Перейдите в папку с проектом
2. Выделите следующие файлы и папки:
   - `manifest.json`
   - `background.js`
   - `content_script.js`
   - `interceptor.js`
   - `bridge.js`
   - `offscreen.html`
   - `offscreen.js`
   - `popup.html`
   - `popup.js`
   - `popup.css`
   - `icon128.png`
3. Нажмите "Сжать" (или создать ZIP-архив). Назовите его, например, `GC_Downloader_v1.zip`.

### 🛠 Как установить (для ваших пользователей)

Передайте пользователю созданный ZIP-архив и эти шаги:

1. **Распакуйте** архив в любую удобную папку (например, на Рабочий стол).
2. Откройте Chrome и перейдите по адресу: `chrome://extensions/`
3. В правом верхнем углу включите **"Режим разработчика"** (Developer mode).
4. Нажмите кнопку **"Загрузить распакованное расширение"** (Load unpacked).
5. Выберите папку, в которую вы распаковали архив.
6. Расширение появится в списке. Нажмите на иконку "Пазл" в панели инструментов Chrome и **закрепите (pin)** наше расширение.

### 🚀 Важные замечания

- **Обновления**: Чтобы обновить расширение, пользователю нужно будет заменить файлы в той же папке и нажать кнопку "Обновить" (иконка закругленной стрелки) на странице `chrome://extensions/`.
- **Безопасность**: Chrome может раз в несколько дней спрашивать: "Вы хотите оставить расширение в режиме разработчика?". Нужно нажать "Оставить".
